Introduction à Git et GitHub

Comprendre les bases de la gestion de versions et collaborer efficacement.

Qu'est-ce que Git ?

Git est un système de contrôle de version qui permet de suivre les modifications de fichiers dans un projet au fil du temps. Il aide les développeurs à travailler en équipe et à gérer les versions de leur code.

Qu'est-ce que GitHub ?

GitHub est une plateforme d'hébergement de code qui utilise Git. Elle permet de stocker, partager et collaborer sur des projets avec d'autres développeurs.

Commandes de Base

  • git init : Initialiser un nouveau dépôt Git.
  • git add : Ajouter des fichiers au suivi de version.
  • git commit : Sauvegarder les modifications avec un message descriptif.
  • git push : Envoyer les changements vers un dépôt distant (ex : GitHub).
  • git pull : Récupérer les changements depuis le dépôt distant.
  • git clone : Copier un dépôt existant.
